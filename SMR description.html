The latest version is compatible with Experimental (Update 6). Install 2.3.0 if you are using Early Access (Update 5)

(Formerly known as Recipe Copier)

Copy various settings from a building into another by pointing and clicking!



With this mod you can select settings from a machine and replicate it along multiple machines of the same kind. You can also aim to a building to just check how it is currently configured



Find me at Satisfactory Modding Discord server (MarcioHuser#2377)

Supported building types:

  • Extractors

    • Miners (Overclock)
    • Water Extractors (Overclock)
    • Oil Extractors (Overclock)
    • Resource Well Pressurizer (Overclock)

  • Generators

    • Biomass Burner (Overclock)
    • Coal Generators (Overclock)
    • Fuel Generators (Overclock)
    • Nuclear Power Plant (Overclock)

  • Production

    • Smelters (Overclock and Recipe)
    • Foundries (Overclock and Recipe)
    • Constructors (Overclock and Recipe)
    • Assemblers (Overclock and Recipe)
    • Manufactorers (Overclock and Recipe)
    • Refineries (Overclock and Recipe)
    • Packagers (Overclock and Recipe)
    • Blenders (Overclock and Recipe)
    • Any MOD machine that accepts overclock and/or use recipes

  • Signs

    • Colors
    • Icons
    • Texts
    • Emissive
    • Glossiness
    • Layout - restricted for signs that share the same subset of layouts

  • Trains

    • Timetables
    • load/unload filters
    • stop type
    • minimum wait time

How to use it:

Unlock it by purchsing the Milestone at Tier 1. The tool can be built at the Equipment Workshop and with that you can:
  • See the information from the building you are aiming;
  • Select the information from the building by clicking on it while holding Left Ctrl;
  • Define the selected information into new compatible buildings by clicking while aiming on it. The tool will glow green if the information can be defined on the building; red if not. It will be off if the building does does not accept the information or already has the same information defined;
  • For recipe and overclock copying you can cycle modes (Recipe & Overclock -> Recipe Only -> Overclock Only) by Rick Clicking the mouse
  • For signs copying you can enable/disable the various settings by doing one of the following:
    • Right Click: enable/disable colors copying
    • Right Click+Left Control: enable/disable colors copying
    • Right Click+Left Alt: enable/disable icons copying
    • Right Click+Left Ctrl+Left Shift: enable/disable emissive and glossiness copying
    • Right Click+Left Shift: enable/disable texts copying
    • Right Click+Left Shift+Left Alt: re-enable all copying settings
